Shortcuts

- Ctrl + S — Save

- Ctrl + Z — Undo

- Ctrl + M — Toggle Multi-Frame Selection

- Ctrl + E — Select Eraser

- Ctrl + P — Select Pen

- Ctr + F - Use Fill Tool

- Left / Right Arrow — Switch Frames
